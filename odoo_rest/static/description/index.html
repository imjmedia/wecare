Odoo Rest API

Now modify your data using HTTP Requests!!

Odoo REST API Module creates a RESTful API for Odoo and allows you to access and modify data using HTTP requests. The admin can fetch, create, update and delete a record from the database by sending the appropriate request in JSON. The response can be chosen to be in either JSON or XML format.

The RESTful API in Odoo!!

A REST API is an application program interface (API) that uses HTTP requests to GET, PUT, POST and DELETE data.

Now, access and modify data using HTTP requests in Odoo. You can easily modify a record from the database as per your wish. Send a request in JSON and choose the response in either JSON or XML format.




Highlighted features


You can easily create API keys to access Endpoints.



A Separate API key is generated for end point users.


You can give different access rights to multiple API keys.



Create and fetch a record from Database.



You can update or even delete a record in Database.


Customize App and Export APK/IPA



Get the Schema of a Table.



Trigger your odoo actions.


View the access rights message in the Endpoint response.

Access and modify data using HTTP requests in Odoo

  • The Odoo REST API Module allows you to access and modify data using HTTP requests.

  • You can create, fetch, update and delete a record from the database by sending the appropriate request in JSON.

Choose the response format in Odoo

  • The response format can be chosen to be in either JSON or XML format.

  • The schema of the table can also be requested from the database.

Create API Key to access the endpoints

  • You can create an API key for accessing the endpoints and set specific data access rights for different users.

  • Separate API keys can be made for different endpoint users.

Manage the API Key and it's access rights

  • You can even restrict specific access rights to different API key users.

  • You can also view the access rights message in the Endpoint response.
