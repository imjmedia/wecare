Sale Order Manager Approval

This Module will allow only the sale manager to approve the sale order if the sale order amount exceeds the minimum amount

Enable Sale Order Approval and Type the Minimum Amount in the Sales Settings

Sale User Creates a Quotation with the total amount that exceeds the minimum amount

While the Sale User Confirms the Quotation, Stage will Change to Approve and The Manager Will need to approve or cancel the record

Now the Sale Manager Can Approve the Record
